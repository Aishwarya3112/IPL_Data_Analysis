# IPL Data Analysis

## Overview
This project analyzes IPL match and ball-by-ball data using Python and Pandas. The objective is to extract meaningful insights from IPL history through data analysis and visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Team with the most wins
- Orange Cap (Highest Run Scorer)
- Purple Cap (Highest Wicket Taker)
- Bowler who conceded the most runs
- Batsman with the most fours
- Batsman with the most sixes
- Toss analysis
- Venue analysis
- Match-winning trends
- Data visualizations using bar charts and pie charts

## Files
- `IPL Analysis.ipynb` – Complete notebook containing code and visualizations.

## Learning Outcomes
- Data cleaning and preprocessing
- GroupBy operations
- Aggregation and sorting
- Data visualization
- Exploratory Data Analysis (EDA)
